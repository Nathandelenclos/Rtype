15 ## Introduction
16 
17 For this Advanced C ++ Knowledge Unit project, R-Type will present the development of network video games.
18 Implementation of a multi-threaded server and a graphical client, using a reusable game engine of our own design.
19 The game should also be fun to play!
20 The purpose of this project is to create a one-to-four player game, using a client/server architecture.
21 This is important. It MUST be a client/server architecture. Peer-to-peer communication is not allowed.
22 
23 ## Goal of the project
24 
25 The goal of this project is to create a multi-player video game based on the rules of the R-Type game.
26 The game must be playable in local and network mode.
27 The game must be composed of a server and a client.
28 The server must be able to manage several simultaneous game rooms.
29 The client must be able to display the game in real time.
30 The client and the server must be able to communicate using the UDP protocol.
31 The server and the client must be able to run on multiple operating systems (Windows, Linux, MacOS).
